# End-to-End Data Analysis of Global Sales Trends 📊

A comprehensive data analysis project using **Python**, **Pandas**, and **Seaborn** to uncover business insights from a Global Superstore dataset.

## 📌 Project Overview
The goal of this project was to analyze raw sales data to identify key performance drivers. I performed data cleaning, handled missing values, and created visualizations to represent sales trends across different categories and regions.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas (Data Manipulation), Matplotlib & Seaborn (Data Visualization)
* **Environment:** VS Code / Jupyter Notebook

## 🧹 Data Cleaning Process
To ensure data accuracy, I performed the following steps:
- **Normalization:** Converted all column headers to lowercase and removed special characters.
- **Handling Missing Values:** Filled missing `postal_code` data with placeholders.
- **Date Formatting:** Converted date strings into Python `datetime` objects for time-series analysis.

## 📈 Key Insights
- **Top Category:** The 'Technology' category consistently drives the highest sales volume.
- **Regional Performance:** Identified the top-performing regions through distribution analysis.
- **Sales Trends:** Visualized annual growth patterns to see peak sales periods.

## 🚀 How to Run
1. Clone the repository.
2. Ensure you have `pandas`, `matplotlib`, and `seaborn` installed.
3. Run `python project.py`.
